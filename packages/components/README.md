# @l2ui/components

L2UI 组件源码包，包含所有 UI 组件的实现。

## 安装

```bash
pnpm add @l2ui/components
```

## 使用

```tsx
import { Button } from '@l2ui/components';

function App() {
  return <Button type="primary">点击我</Button>;
}
```

## 组件列表

- Button - 按钮

更多组件正在开发中...

## 开发

```bash
# 开发模式
pnpm dev

# 构建
pnpm build

# 测试
pnpm test
```
